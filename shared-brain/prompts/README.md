# 📝 Shared Prompts

*Useful prompts that work well. Share your discoveries.*

---

## 🔥 FULL CODE Activation

When complex code is needed:

```
🔥 FULL CODE ACTIVATED 🔥
Task: [description]
Pipeline: Codex → Claude Code → Implement
```

---

## 🐝 Swarm Deployment

For parallel work across multiple sub-agents:

```
Deploy a swarm of [N] agents:
- Agent 1: [specific task]
- Agent 2: [specific task]
- ...

Each agent should:
1. Work independently
2. Report results to shared location
3. Not duplicate effort

Coordination: [how they'll sync]
```

---

## 📊 QA Verification

After building something:

```
Deploy QA verification:
1. Spawn [N] QA sub-agents
2. Each reviews different modules
3. Run all tests
4. Fix bugs found
5. Report: bugs fixed, tests created, coverage

Target: [project path]
```

## 🏗️ Project Scaffolding

For new projects:

```
Create project structure for [name]:
1. Directory layout
2. Core modules
3. Config files
4. README.md
5. Tests folder
6. Dashboard page
7. Add to SOUL.md
8. Add to shared-brain
```

## 🔍 Memory Search

When context is lost:

```
Search memory for [topic]:
1. Check CURRENT_STATE.md
2. Search memory/*.md
3. Check SOUL.md sections
4. Check shared-brain/
5. Summarize what you find
```
