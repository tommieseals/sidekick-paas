## Microsoft Graph Integration Details

The new `microsoft_graph.py` provides:
- **Outlook:** List, send, reply, move emails
- **Calendar:** List, create, delete events; Teams meetings
- **OneDrive:** List, upload, download files; sharing links
- **Excel:** Read/write ranges, create tables
- **SharePoint:** Sites, lists, items
- **Teams:** Channels, messages, online meetings
- **Users:** Profile info, photos

Uses OAuth2 client credentials flow - works with same Azure AD creds.
