# 2026-03-01 - 2Captcha Cloudflare Turnstile Fix WORKING

## Major Breakthrough: Indeed CAPTCHA Bypass

**Problem:** Indeed uses Cloudflare Challenge Pages which require extra parameters (action, data, pagedata) that 2Captcha's basic Turnstile endpoint rejects with ERROR_BAD_PARAMETERS.

**Solution Found:**

### 1. Intercept turnstile.render() BEFORE page loads
```javascript
window.turnstile.render = function(element, params) {
    window.__turnstileData = {
        sitekey: params.sitekey,
        action: params.action,           // "managed" for Challenge Pages
        cData: params.cData,             // e.g. "9d5d9a45a8bfd925"
        chlPageData: params.chlPageData  // ~500 char token with timestamp
    };
    window.__turnstileCallback = params.callback;
    return originalRender.call(this, element, params);
};
```

### 2. Use 2Captcha createTask API (NOT in.php)
```json
POST https://api.2captcha.com/createTask
{
  "clientKey": "YOUR_KEY",
  "task": {
    "type": "TurnstileTaskProxyless",
    "websiteURL": "https://www.indeed.com/jobs",
    "websiteKey": "0x4AAAAAAADnPIDROrmt1Wwj",
    "action": "managed",
    "data": "9d5d9a45a8bfd925",
    "pagedata": "CBhnaO8EZ3EY6l7MpJ..."
  }
}
```

### 3. Inject token via captured callback
```javascript
window.__turnstileCallback(token);
```

## Key Learnings

1. **Headless = Hard Block** - Indeed hard-blocks headless browsers (no CAPTCHA, just "Request Blocked")
2. **Non-headless = Solvable CAPTCHA** - Shows Turnstile checkbox that 2Captcha can solve
3. **pagedata expires quickly** - Must capture fresh params and submit immediately
4. **First attempt may fail** - ERROR_CAPTCHA_UNSOLVABLE sometimes, retry works
