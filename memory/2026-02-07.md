### Challenges & Fixes

**Issue 1: Low RAM (2.5GB free)**
- **Action:** Proactively closed Chrome to free memory
- **Lesson:** Be proactive with resource management

**Issue 2: RAM check script exiting parent scripts**
- **Fix:** Updated `get_free_ram.sh` to only exit when run directly, not when sourced
- **Pattern:** Check if `${BASH_SOURCE[0]} == ${0}` before exiting

**Issue 3: JSON escaping in jq**
- **Root cause:** Manually building JSON with string interpolation
- **Fix:** Use jq's `--arg` parameters exclusively, let jq handle all escaping
- **Working pattern:**
  ```bash
  RESULT=$(jq -n \
    --arg ts "$TIMESTAMP" \
    --arg ram "$free_ram" \
    --arg analysis "$ANALYSIS" \
    '{timestamp:$ts, ram_mb:($ram|tonumber), analysis:$analysis}')
  ```

**Issue 4: Whitespace in numeric values**
- **Fix:** Added `tr -d ' \n'` to clean wc output
- **Example:** `wc -l | tr -d ' \n'`
